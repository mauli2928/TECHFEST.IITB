# Techfest IIT Bombay — Cyborg Future Landing Page

Ultra-premium, fully responsive cinematic landing page for **Techfest IIT Bombay** — Asia's Largest Science & Technology Festival.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- **Real Techfest content** — stats, competitions, Robowars, TechConnect, dates (Dec 22–24, 2025), official links
- **Vibrant futuristic palette** — electric blue, cyan, ultraviolet purple, cinematic gradients
- **10 sections** — Hero, About, Evolution scroll, Technology, 3D Experience, Events HUD, Community, CTA, Command-center footer
- **Animations** — GSAP ScrollTrigger, Framer Motion, Lenis smooth scroll, scroll progress bar
- **3D** — React Three Fiber neural core (mouse + scroll reactive, mobile optimized)
- **Responsive** — mobile-first, no horizontal overflow, adaptive 3D/particle density

## Stack

Next.js 16 · TypeScript · Tailwind v4 · Framer Motion · GSAP · Lenis · R3F · ShadCN-style UI
