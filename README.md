# TECHFEST.IITB
Techfest IIT Bombay Cyborg Future Experience Techfest IITB is a futuristic, fully responsive, cinematic web experience inspired by AI, cybernetic evolution, robotics, and next gen tech ecosystems. The project reimagines how a global tech fest should look &amp; feel in the modern era immersive, interactive, visually powerful &amp; technologically advanced.
